# Site Map — KINEXIS Digital

Route, purpose, and section breakdown for every page. Matches the page
stubs already created under `src/app/`.

## 1. Homepage — `/`

**Purpose:** sell trust.

| Section | Notes |
|---|---|
| Hero | Headline: "Growth That Leaves A Lasting Mark." Subheadline: "Strategic websites, SEO, and digital marketing built to generate measurable business growth." Buttons: "Book Consultation" (primary), "View Work" (secondary). |
| Trusted Process | Simple 4-step process — define KINEXIS's actual 4 steps. |
| Services Overview | Cards: Website Design, SEO, Google Ads, Local SEO, Branding. |
| Featured Results | Case studies with actual numbers. |
| Why KINEXIS | Differentiators. |
| CTA | Schedule a strategy call. |

## 2. Services — `/services`

Overview of all services, cards linking to each individual service page.

## 3. Individual service pages

| Service | Route |
|---|---|
| Website Design | `/services/web-design` |
| SEO | `/services/seo` |
| Google Ads | `/services/google-ads` |
| Local SEO | `/services/local-seo` |
| Branding | `/services/branding` |

Each page: Benefits → Process → FAQ → CTA.

## 4. Case Studies — `/case-studies`

This is where agencies win clients. Index page lists all case studies;
`/case-studies/[slug]` is the individual case study template.

Each case study: Problem → Strategy → Solution → Results.

Example from the brief: landscaping company, $2,000/month before →
$11,500/month after.

## 5. About — `/about`

Mission, Values, Process, Philosophy. Not "we are passionate" copy —
explain the actual reasoning behind how KINEXIS works.

## 6. Blog — `/blog`

SEO content engine. Categories: SEO, Web Design, Marketing, Google Business
Profile, Local Business Growth. `/blog/[slug]` is the individual post
template.

## 7. Contact — `/contact`

Form, Calendly embed, direct email.

---

## Global elements (every page)

- **Header:** full logo, primary nav (Home, Services, Case Studies, About,
  Blog, Contact), "Book Consultation" button.
- **Footer:** icon logo, nav links, services list, contact info, socials,
  copyright.
